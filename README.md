## 参考文档
- https://llvmlite.readthedocs.io/en/latest/


## 环境配置:

安装 https://www.antlr.org/

安装antlr4-python3-runtime

``` 
pip install antlr4-python3-runtime
```

安装llvmlite

```
pip install llvmlite
```


## 运行

MACOS
```
/usr/local/opt/llvm/bin/lli xxx.ll
```
